Trading insights

Dashboard

Tokemon provides real time trading insights for the ERC20 tokens on the decentralized exchanges (such as Uniswap) through a powerful dashboard. The data is presented to you as soon as it's validated on the blockchain and processed by our backend. The dashboard features the following data per token:

  • Latest price
  • Liquidity
  • Token pool reserves
  • ETH pool reserves
  • Holders count
  • Price incline/decline trend
  • Contract verification data

Trading notifications/Social platform bots

The other core insights feature that Tokemon provides is notifying you when a certain trading event occurs, specified by the configuration. You're able to set a global configuration (rules for all tokens) and configuration per token.

Global configuration

Options for the global configuration:

  • Report interval (in minutes): You will get notified if some of the token criteria were met in the last X minutes.
  • Percent increase: You will get notified when the token value increases by the selected percent.
  • Percent decrease: You will get notified when the token value decreases by the selected percent.
  • Minimum liquidity: You will get notified only if the token contains the minimum liquidity in the pool.
  • Contract verified: You will get notified only if the token's contract is verified.

Per token configuration

Options for the token configurations:

  • Report interval (in minutes): You will get notified if some of the token criteria were met in the last X minutes.
  • Percent increase: You will get notified when the token value increases by the selected percent.
  • Percent decrease: You will get notified when the token value decreases by the selected percent.
  • Target value: You will get notified when the token increases above or decreases below this value.
  • Minimum liquidity: You will get notified only if the token contains the minimum liquidity in the pool.
  • Contract verified: You will get notified only if the token's contract is verified.
